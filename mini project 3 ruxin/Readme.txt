The main program is mini project 3 Ruxin.py

By running on the terminal, you can choose the database type(Mysql/Mongodb).
If you choose to search the API(function was written in search_api.py), you can get the images with one selected tag.
Also if you choose to show the database information(function was written in show_db_info.py), you can know how many images are in the database, and get the most frequent tag.
